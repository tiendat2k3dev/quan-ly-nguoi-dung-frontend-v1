# User Management System

[![Demo](https://img.shields.io/badge/-Live%20Demo-brightgreen)](https://quan-ly-nguoi-dung-frontend-v1.vercel.app/)

## 📱 Overview
Hệ thống quản lý người dùng (User Management System) - Frontend application cho việc quản lý tài khoản người dùng, phòng ban, xác thực và cập nhật thông tin.

### Thư viện chính
- **Quản lý state**: React Context API
- **UI**: Ant Design
- **Routing**: React Router DOM
- **Gọi API**: Axios
- **Thông báo:**: React Toastify
- **Icons**: FontAwesome, React Icons

## ✨ Features
- ✅ User account management (CRUD, pagination, filter by role/dept)
- ✅ Department management (CRUD)
- ✅ Authentication (Login, Forgot Password, Verification)
- ✅ Profile update & Password change
- ✅ Responsive design with Ant Design
- ✅ Protected routes

## 👨‍💻 My Role
- Developed responsive UI components using React and Ant Design.
- Implemented state management with Redux Toolkit.
- Integrated REST API with Axios for user/department data.
- Built authentication flow (login, token handling, protected routes).
- Designed modals for add/edit/delete/reset operations.
- Deployed to Vercel with CI/CD.
