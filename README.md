 User Management System (React + Spring Boot)
🚀 Live Demo: https://web-quan-ly-nguoi-dung-front-end.vercel.app

Giới thiệu
Dự án xây dựng hệ thống quản lý người dùng (User Management System) với các chức năng:
Quản lý tài khoản (CRUD)
Quản lý phòng ban (CRUD)
Xác thực người dùng (JWT + refresh token)
Quên mật khẩu bằng OTP

Tính năng chính:
Authentication
Đăng nhập / Đăng xuất
JWT + Refresh Token (auto refresh)
Protected Routes
Quên mật khẩu + OTP
Đổi mật khẩu
Account Management
Thêm / Sửa / Xóa user
Phân quyền (ADMIN, EMPLOYEE, MANAGER)
Filter theo role / department
Pagination
 Department Management
CRUD phòng ban
Filter theo ngày
Paginatio

Tech Stack
Frontend:
React 18, Vite
Ant Design
React Router v6
Axios (Interceptor)
